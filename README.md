Answer of all question:

1.JSX: A syntax that lets you write HTML-like code inside JavaScript.

2.State vs Props: State is local and changeable; props are read-only and passed from parent.

3.useState hook: A hook to add and update state in functional components.

4.Share state: Pass it as props or lift state up to a common parent.

5.Event handling: Use React events like onClick with a handler function.
